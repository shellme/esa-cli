---
title: "初期設定"
description: "esa-cliの初期設定を行うコマンド"
---

esa-cliの初期設定を行います。このコマンドは、チーム名とアクセストークンを設定し、設定ファイルを生成します。

## 仕様

### コマンド形式

```bash
esa-cli setup
```

### オプション

- `-team` - チーム名を指定
- `-token` - アクセストークンを指定

### 設定ファイル

このコマンドは `~/.esa-cli-config.json` に設定ファイルを生成します。

```json
{
  "team_name": "your-team-name",
  "access_token": "your-access-token"
}
```

## 使用例

### 基本的な設定

```bash
# 対話形式で設定
esa-cli setup

# オプションを指定して設定
esa-cli setup -team "your-team-name" -token "your-access-token"
```

### 設定の確認

```bash
# 設定ファイルの内容を確認
cat ~/.esa-cli-config.json
```

### 設定の更新

```bash
# 設定を更新
esa-cli setup -team "new-team-name" -token "new-access-token"
```

## 注意事項

- アクセストークンは機密情報です。設定ファイルのパーミッションを適切に設定してください。
- チーム名は、esa.ioのURLに含まれるチーム名を指定します。
- 設定ファイルは `~/.esa-cli-config.json` に保存されます。
