esa-cliの初期設定を行います。チーム名とアクセストークンを対話的に入力して設定ファイルを生成します。

## 仕様

### コマンド形式

```bash
esa-cli setup
```

### 動作

- チーム名（サブドメイン）とアクセストークンを対話的に入力
- 設定が自動的にテストされ、接続確認が行われます
- 設定ファイル `~/.esa-cli-config.json` を生成します

<Aside type="tip" title="初回使用時は必須">
このコマンドは、esa-cliを使用する前に必ず実行してください。設定が完了していないと他のコマンドは動作しません。
</Aside>

### 設定ファイル

このコマンドは `~/.esa-cli-config.json` に設定ファイルを生成します。

```json
{
  "team_name": "your-team-name",
  "access_token": "your-access-token"
}
```

## 使用例

### 基本的な設定

```bash
# 対話形式で設定
esa-cli setup
```

### 設定の確認

```bash
# 設定ファイルの内容を確認
cat ~/.esa-cli-config.json
```

## 注意事項

<Aside type="caution" title="セキュリティに注意">
アクセストークンは機密情報です。設定ファイルのパーミッションを適切に設定してください。
</Aside>

- チーム名は、esa.ioのURLに含まれるチーム名を指定します。
- 詳細な設定手順については[インストールガイド](/esa-cli/getting-started/installation)を参照してください。
