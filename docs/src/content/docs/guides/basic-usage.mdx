初めての方でも迷わず使えるよう、よく使うコマンドとそのユースケースを紹介します。

## 記事の一覧表示

最新の記事を一覧表示します。
```bash
esa-cli list
```

カテゴリやタグで絞り込みも可能です。
```bash
esa-cli list -c "開発/ドキュメント"
esa-cli list --category "開発/ドキュメント"
esa-cli list -t "API"
esa-cli list --tag "API"
```

## 記事の検索

タイトルや本文にキーワードを含む記事を検索できます。
```bash
esa-cli list -q "検索キーワード"
esa-cli list --query "検索キーワード"
```

## 作成者での絞り込み

自分が作成した記事を絞り込んで表示できます。
```bash
esa-cli list -u "自分のユーザー名"
esa-cli list --user "自分のユーザー名"
```

## 記事のダウンロード

記事番号を指定してMarkdownファイルとしてダウンロードします。
```bash
esa-cli fetch 123
```

フィルタリングと組み合わせて最新記事をダウンロードできます。
```bash
esa-cli fetch -c 開発 -l
esa-cli fetch --category 開発 --latest
```

## 記事の更新

ローカルで編集したMarkdownファイルから記事を更新できます。
```bash
esa-cli update 123-article-title.md
```

オプションを組み合わせて効率的に更新できます。
```bash
esa-cli update 123-article-title.md -c 開発 -m "カテゴリを変更"
esa-cli update 123-article-title.md --category 開発 --message "カテゴリを変更"
```

## 記事の一括移動

特定の条件で記事を一括でカテゴリ移動できます。
```bash
esa-cli move -c 開発 -o デザイン -u 自分のユーザー名
esa-cli move --category 開発 --to デザイン --user 自分のユーザー名
```

## リモート変更の安全な更新

チームで記事を編集する際、他の人が更新した内容を誤って上書きすることを防ぐ機能があります。

```bash
# 更新時にリモート変更があれば警告が表示されます
esa-cli update 123-article-title.md
```

**警告が表示された場合：**
- リモート記事の最終更新日時が表示されます
- 上書きするかどうか確認できます
- `y`で更新、`N`でキャンセルが選択できます

## カテゴリ・タグの活用

記事作成や更新時に `-category` や `-tag` オプションを使うことで、記事を整理できます。

## ユースケース

- **ドキュメント管理**: プロジェクトの設計書やAPI仕様をesaで一元管理
- **ナレッジ共有**: チームのTipsや手順書を素早く投稿・更新
- **定期バックアップ**: 記事を定期的にローカルへダウンロード

詳細なコマンド仕様や高度な使い方は[コマンドリファレンス](/commands/)や[高度な使い方](/guides/advanced-usage)を参照してください。
