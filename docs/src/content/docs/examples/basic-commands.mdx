## 📝 記事の操作

### 記事の一覧表示

記事の一覧表示は、esaの記事を効率的に管理するための基本的な機能です。特定のカテゴリやタグの記事を素早く見つけたり、最近の更新を確認したりするのに役立ちます。

```bash
# 最新の記事を表示
# 最近の更新を素早く確認したい場合に便利です
esa-cli list

# 特定のカテゴリの記事を表示
# プロジェクトやチームごとの記事を整理して表示できます
esa-cli list --category "開発/ドキュメント"

# 検索条件を指定して表示
# 特定のキーワードを含む記事を素早く見つけられます
esa-cli list --query "重要"
```

詳細は[ユーザーガイドの記事一覧表示](/guides/user-guide/#記事の一覧表示)を参照してください。

### 記事の表示

記事の表示機能は、esaの記事を様々な形式で確認するのに役立ちます。Markdown形式で表示することで、記事の構造を確認したり、HTML形式で表示することで、ブラウザでの表示を確認したりできます。

```bash
# 記事番号を指定して表示
# 特定の記事の内容を素早く確認できます
esa-cli show 123

# 記事をMarkdown形式で表示
# 記事の構造やフォーマットを確認するのに便利です
esa-cli show 123 --format markdown

# 記事をHTML形式で表示
# ブラウザでの表示を確認するのに役立ちます
esa-cli show 123 --format html
```

詳細は[ユーザーガイドの記事表示](/guides/user-guide/#記事の表示)を参照してください。

### 記事の作成

記事の作成機能は、esaに新しい記事を効率的に追加するのに役立ちます。カテゴリの指定やテンプレートの使用により、一貫性のある記事を作成できます。

```bash
# 新規記事を作成
# 基本的な記事作成に使用します
esa-cli create "新しい記事" --body "# 本文"

# カテゴリを指定して作成
# 記事を適切なカテゴリに分類できます
esa-cli create "新しい記事" --category "開発/ドキュメント" --body "# 本文"

# テンプレートを使用して作成
# 一貫性のある記事構造を維持できます
esa-cli create "新しい記事" --template "テンプレート名"
```

詳細は[ユーザーガイドの記事作成](/guides/user-guide/#記事の作成)を参照してください。

## 🔄 記事の更新

記事の更新機能は、既存の記事を効率的に編集するのに役立ちます。タイトル、本文、カテゴリなど、様々な要素を更新できます。

```bash
# 記事の更新
# 記事の内容を修正する場合に使用します
esa-cli update 123 --body "# 更新された本文"

# タイトルの更新
# 記事のタイトルを変更する場合に使用します
esa-cli update 123 --title "更新されたタイトル"

# カテゴリの更新
# 記事の分類を変更する場合に使用します
esa-cli update 123 --category "新しいカテゴリ"
```

詳細は[ユーザーガイドの記事更新](/guides/user-guide/#記事の更新)を参照してください。

## 📥 記事のダウンロード

記事のダウンロード機能は、esaの記事をローカルに保存したり、バックアップを作成したりするのに役立ちます。特定のディレクトリに保存することで、記事を整理して管理できます。

```bash
# 記事をMarkdown形式でダウンロード
# 記事をローカルで編集する場合に使用します
esa-cli download 123

# 特定のディレクトリにダウンロード
# 記事を整理して保存する場合に使用します
esa-cli download 123 --output ./docs

# 記事番号を指定してダウンロード
# 複数の記事を一度にダウンロードする場合に使用します
esa-cli download 123 456 789
```

詳細は[ユーザーガイドの記事ダウンロード](/guides/user-guide/#記事のダウンロード)を参照してください。

## 🔍 検索

検索機能は、esaの記事を効率的に見つけるのに役立ちます。キーワード、カテゴリ、タグなど、様々な条件で検索できます。

```bash
# キーワードで検索
# 特定の内容を含む記事を探す場合に使用します
esa-cli search "検索キーワード"

# カテゴリを指定して検索
# 特定のカテゴリ内で記事を探す場合に使用します
esa-cli search "検索キーワード" --category "開発"

# 検索結果をMarkdown形式で表示
# 検索結果を保存したり、他のツールで利用する場合に使用します
esa-cli search "検索キーワード" --format markdown
```

詳細は[ユーザーガイドの検索](/guides/user-guide/#検索)を参照してください。
