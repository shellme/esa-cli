# コマンド例

esa-cliの様々な使用例を紹介します。基本的な使い方から高度な使い方まで、目的に応じて参照してください。

## 基本的なコマンド例

基本的なコマンドの使用例は[基本的なコマンド例](/examples/basic-commands)を参照してください。

- 記事の一覧表示
- 記事のダウンロード
- 記事の更新

## 高度なコマンド例

より高度な使い方や便利な組み合わせは[高度なコマンド例](/examples/advanced-commands)を参照してください。

- 便利な組み合わせ例
- 一括処理の例
- シェルスクリプトの例
  - 記事の一括バックアップ
  - カテゴリの一括移動
  - 記事の定期バックアップ

## カスタマイズのヒント

これらの例は、必要に応じてカスタマイズして使用できます。例えば：

- エディタの設定を変更する
- バックアップの頻度を調整する
- 通知の内容をカスタマイズする
- シェルスクリプトを組み合わせる
