esa-cliは以下の2つの方法でインストールできます。最も簡単な方法はHomebrewを使用する方法です。

## 方法1: Homebrewを使用（推奨）

### 前提条件
- macOS または Linux
- Homebrewがインストールされていること

### インストール手順

```bash
brew tap shellme/esa-cli
brew install esa-cli
```

### 動作確認

インストールが成功したか確認するには：

```bash
esa-cli --version
```

## 方法2: ソースコードからビルド

### 前提条件
- Go 1.16以上
- Git
- ビルドツール

### インストール手順

<Steps>

1. リポジトリをクローン

   ```bash
   # リポジトリをクローン
   git clone https://github.com/shellme/esa-cli.git
   cd esa-cli
   ```

2. ビルド

   ```bash
   # ビルド
   go build -o esa-cli
   ```

3. ビルドしたバイナリをパスの通ったディレクトリに移動

   ```bash
   # ビルドしたバイナリをパスの通ったディレクトリに移動
   sudo mv esa-cli /usr/local/bin/
   ```

</Steps>

## トラブルシューティング

### インストール時のエラー

インストール時にエラーが発生した場合は、以下の手順を試してください：

<Steps>

1. 既存のインストールを削除

   ```bash
   # 既存のインストールを削除
   brew uninstall esa-cli
   ```

2. 再インストール

   ```bash
   # 再インストール
   brew tap shellme/esa-cli
   brew install esa-cli
   ```

</Steps>

### コマンドが見つからない場合

`esa-cli` コマンドが見つからない場合は、以下を確認してください：

<Steps>

1. パスが正しく設定されているか確認
   ```bash
   # パスが正しく設定されているか確認
   echo $PATH
   ```

2. `/usr/local/bin` がパスに含まれているか確認
   ```bash
   # `/usr/local/bin` がパスに含まれているか確認
   echo $PATH | grep /usr/local/bin
   ```

3. 必要に応じて、使用しているシェルの設定ファイルにパスを追加
   ```bash
   # bashの場合（`~/.bashrc` または `~/.bash_profile`）:
   export PATH="/usr/local/bin:$PATH"
   ```

   ```bash
   # zshの場合（`~/.zshrc` または `~/.zprofile`）:
   export PATH="/usr/local/bin:$PATH"
   ```

   ```bash
   # fishの場合（`~/.config/fish/config.fish`）:
   set -gx PATH /usr/local/bin $PATH
   ```
</Steps>
