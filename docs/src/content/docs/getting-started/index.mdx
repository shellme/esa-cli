esa-cliは、esa.ioの記事をコマンドラインから操作するためのツールです。このガイドでは、インストールから基本的な使い方まで、順を追って説明します。

## 前提条件

- macOS または Linux 環境
- インターネット接続
- esa.ioのアカウント
- ターミナルでの基本的な操作知識
- 以下のいずれかのインストール方法が利用可能：
  - Homebrew（macOSの場合）またはLinuxbrew（Linuxの場合）
    - インストール方法: [Homebrew公式サイト](https://brew.sh/)
  - または、バイナリファイルを直接ダウンロードしてインストール
    - 詳細は[インストール](/getting-started/installation)を参照

## クイックスタート

<Steps>

1. インストール（Homebrewの場合）
   ```bash
   brew install shellme/esa-cli
   ```

2. 初期設定
   ```bash
   esa-cli setup
   ```

3. 記事の一覧表示
   ```bash
   esa-cli list
   ```

4. 記事のダウンロード
   ```bash
   esa-cli fetch <記事番号>
   ```

</Steps>

## 主な機能

- 記事の一覧表示と検索
- 記事のダウンロードと更新
- ローカルでの編集と同期
- 通知機能
- Spotlight検索との連携
